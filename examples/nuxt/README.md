# Arcjet + Nuxt

This example shows how to use Arcjet in a
[Nuxt](https://nuxt.com/) app.

## Use

Clone the monorepo, then:

```sh
cd arcjet-js
npm ci
cd examples/nuxt/
npm ci
```

Rename `.env.example` to `.env` and add your Arcjet key.
Then:

```sh
npm run dev
```

Now visit `http://localhost:3000/` in a browser and follow the steps
explained there.
